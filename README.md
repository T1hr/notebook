# notebook
集日历查询，白噪音，计时器，记事本为一体的Android开发App

## Features

### 日程安排
本app是一款管理待办事项，记录日程计划的效率类应用，主要实现日程安排，事件记录，笔记复盘，任务倒计时，音乐播放等功能。软件页面如下：
